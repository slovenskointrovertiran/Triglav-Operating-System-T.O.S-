# Triglav Operating System (T. O. S.)

An Operating System made in C and NASM for x86 architecture, 32 bits protected mode. 

This operating system is currently in a state of developpement. 

Everybody who wants to see how TriglavOS works detain the right to look at the code. However, forks and modifications are forbidden.
If you have some advices, contact me.

What will TriglavOS have ?

- A full shell insipired by UNIX
- A file system (TFS)
- Full support of english, french and slovene languages.
- AZERTY, QWERTY, QWERTZ keyboards
- I hope one day a graphical environnement.
